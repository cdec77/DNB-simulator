# DNB-simulator
Saisir les notes :

Remplissez vos notes pour chaque trimestre (T1, T2, T3) dans les cases correspondantes.
Si une note est manquante, laissez la case vide : la moyenne sera calculée sur les notes disponibles.


Option Latin :

Si vous suivez l'option Latin, remplissez vos notes.
Seuls les points au-dessus de 10/20 seront ajoutés à votre moyenne de contrôle continu.


Simuler les épreuves finales :

Entrez vos notes potentielles pour les 5 épreuves finales (Français, Maths, Histoire-Géo/EMC, Sciences, Oral).
La moyenne des épreuves finales est calculée automatiquement.


Visualiser les résultats :

Votre moyenne de contrôle continu (40%) s'affiche en temps réel.
Votre moyenne des épreuves finales (60%) s'affiche en temps réel.
Votre moyenne finale du DNB est calculée.
La mention obtenue (ou les points manquants) s'affiche instantanément.

Exemple :

Moyenne de contrôle continu : 14/20 (avec +2 points de Latin).
Moyenne des épreuves finales : 12/20.
Moyenne finale : (14 × 0.4) + (12 × 0.6) = 12.8/20 → Mention "Assez bien".

5.2. Pour les enseignants


Utilisation en classe :

Projetez le simulateur pour expliquer le calcul des moyennes.
Montrez comment les notes des trimestres impactent le résultat final.


Suivi individuel :

Demandez aux élèves de remplir leurs notes et de simuler différents scénarios.
Utilisez le simulateur pour identifier les matières à améliorer.


Préparation aux épreuves :

Simulez des notes aux épreuves finales pour montrer aux élèves ce qu'ils doivent viser.
Discutez des stratégies pour atteindre les mentions souhaitées.


Intégration dans les outils pédagogiques :

Ajoutez le lien vers le simulateur dans votre ENT ou site de classe.
Utilisez-le comme outil de révision avant les épreuves.
